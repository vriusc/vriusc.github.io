Windows环境下SQL2019的下载与安装

一.安装版本：

1.SQL Sereve 2019 CTP(Community Test Preview 社区测试试用版 )

下载地址:
https://www.microsoft.com/en-us/evalcenter/evaluate-sql-server-2019-ctp


此处使用的是SQL sercer的试用版本，建议使用以前发行的RTM（零售版）版本
下载地址：
https://msdn.itellyou.cn/
打开网址单机服务器选择你想要的SQL版本并下载所需的版本。
下载的文件应该为iso镜像文件。

2.Microsoft SQL Server Management Studio 18

下载网址：
https://docs.microsoft.com/zh-cn/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-2017

3.安装 SQL Server 的硬件和软件要求


内存和处理器要求：
==详细内容请访问:==
https://docs.microsoft.com/zh-cn/sql/sql-server/install/hardware-and-software-requirements-for-installing-sql-server?view=sql-server-ver15

二.安装步骤：

1.SQL Server 2019的安装

1.选择已经下载好的iso文件并将其打开

2.打开后点击安装->全新SQL Server 独立安装或先现有安装添加功能。

3.进入安装界面按步骤点击下一步知道安装类型（在安装规则时会碰到windows防火墙的警告，不用理会），然后点击执行SQL Server 2019 CTP2.3的全新安装。

4.选择自己所需要的版本，在这里我们选择developer版本。硬盘容量较为紧张的用户也可以选择Express。


5.点击下一步到功能选择界面，选择自己所需的功能，不建议全选很多功能暂时用不上。必选：数据库引擎服务、和SQL复制。（建议直接按下图选择），并选择安装的实例根目录，建议默认安装就好。

6.可以选择默认实例，如需命名，不可重复。
7.默认下一步到数据库引擎配置。选择混合模式，并设置密码。此时用户名为sa,并点击添加当前用户添加用户。
8.默认下一步等待程序安装完成。

2.SSMS的安装

1.点击SSMS0-Setup_CHS.exe,默认安装即可。

3.打开软件并应用。

1.点击开始列表找到Microsoft SQL Server Management studio.点击并打开他。

2.服务器名与上面的实例ID相对应，然后点击点击连接即可。

3.下面就可以使用SQL Server 2019了。（其他版本大致与2019相似，模仿下载即可。）
